将常用的echarts图表封装成angularjs指令

基于echarts3.0，angular1.3.15

支持的功能：

> 1. 最简单的数据渲染 和 样式控制
> 2. 与echarts中完全一致的数据渲染 和 样式控制
> 3. 自定义鼠标点击事件
